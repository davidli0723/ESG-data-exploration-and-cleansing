# ESG data exploration and cleansing

Overview
This project is a platform for managing and analysing Environmental, Social, and Governance (ESG) data. It integrates data ingestion and semantic querying to support ESG metric calculations, making it ideal for users exploring sustainability metrics and reporting frameworks.

This section covers EDA and data cleansing to provide insights for the ESG data platform. The data source is from [Eurofidai](https://www.eurofidai.org/product/esg-raw-data-corporations).
